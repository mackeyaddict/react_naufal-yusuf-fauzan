# Resume Materi Bootstrap

1. Bootstrap adalah framework css gratis, mudah dipelajari, cepat, dan mendukung responsif.

2. Grid system pada Bootstrap membagi layout menjadi 12 kolom, dan memungkinkan konten website ditata secara responsive berdasarkan ukuran layar. Grid system sangat membantu dalam mendesain layout website yang responsive, sehingga tampilan website dapat menyesuaikan ukuran desktop, tablet, atau mobile.

3. Bootstrap library. Ini dapat digunakan untuk membuat tampilan website yang lebih interaktif dengan animasi-animasi yang sudah disediakan.
